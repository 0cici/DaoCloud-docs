# Release 0.4.5

## 功能特性

- 增加告警通知模块主要 API
- 升级并适配 kpanda 0.4.x API
- 为系统日志增加日志所属文件路径信息
- 增加查询单条日志上下文 API
- 增加查询 Kubernetes Event API
- 增强 Insight 自身可观测性能力, 提供自身的指标接口和查询链路信息
- 通过反向代理 Jaeger Query 的 API 供前端使用
- 增加 Query Tracing Operations 相关 API
- 增加 Span Metric 相关 API

## 测试

- 增加 E2E 用例覆盖率徽章
- 补充告警通知相关的测试用例文档
